Requirements

System Requirements:

  1. Supported Linux OS - Ubuntu, RHEL and AWS Linux

  2. Pre-requisites installed

  3. Cloud AI 100 Platform and Apps SDK installed

  4. Multi-device support enabled for model sharding

  • Use bash terminal

  • If using ZSH terminal then “device_group” should be in single quotes e.g. “–device_group [0]”

Installation

Using SDK

  • Donwload Apps SDK: Cloud AI 100 Platform and Apps SDK install

# Install Apps SDK

bash install.sh —enable-qeff

Using GitHub Repository

# Create Python virtual env and activate it. (Required Python 3.8)

python3.8 -m venv qeff_env
source qeff_env/bin/activate
 
# Clone the QEfficient Repo.
https://github.com/quic/efficient-transformers
 
# Install the qefficient-library in Host machine (Used for CLI APIs) (Until we have docker integrated in Apps SDK)
pip install -e .
